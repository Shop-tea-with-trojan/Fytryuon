Язык программирования Fytryuon

Fytryuon — это экспериментальный язык программирования, разработанный для изучения нового синтаксиса и подходов к решению задач. Его особенности:

  • Минималистичный синтаксис
  • Интуитивно понятные конструкции
  • Гибкость для написания как простых, так и сложных программ

Пример кода:

Import_math
Sumbyt_print: "Привет, мир!");
/"Начало блока математики"/
:math_start :

    /"Объявление переменной x и присваивание ей значения 10"/
    Uoion = variable, Variable = x
    Uoion_meaning x, Holdet_meaning, meaning = [ 10 ]

    /"Объявление переменной y и присваивание ей значения 5"/
    Uoion = variable, Variable = y
    Uoion_meaning y, Holdet_meaning, meaning = [ 5 ]

    /"Вычисляем сумму x + y и выводим результат"/
    Uoion = math
        Sumbyt_print: "x + y = "
        Sumbyt_print: [ x + y ]

:math_stop :
